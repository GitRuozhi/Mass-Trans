# Mass-Trans

**Mass-Trans** 是一个面向漫画翻译与本地化的 AI 工作流项目，MVP 以 Codex / Claude Code 本地 Skill 的形式实现。

## 愿景

**Mass-Trans** 希望尽可能降低漫画翻译工作的门槛，同时提升专业级工作的上限。

**Mass-Trans** 希望让新手可以更快完成基本翻译工作，也让有经验的译者、嵌字和修图人员能够在 AI 的辅助下进行更精细的控制。

## 为什么做这个项目

漫画翻译不是简单的 OCR 加机器翻译。一部漫画中存在 连贯剧情、分镜关系、专有名词、角色语言习惯等复杂的相互关系。

当前的翻译工具往往以单页为基础处理，存在画面涂改、前后文断裂、术语不统一等诸多问题。


## 核心能力

### 1. 以整本漫画作为项目处理

**Mass-Trans** 以整本漫画为基本单位，摈弃以**页**为基础的传统工作流。

**Mass-Trans** 会尽可能保留章节、页面、分镜和上下文之间的关系，为后续翻译、校对和回填提供基础。

### 2. 面向分镜的工作流

分镜是漫画叙事的核心单元。Mass-Trans 会优先围绕分镜组织内容，帮助用户和 Agent 更好地理解画面、对白和剧情。

### 3. 区分不同类型的文本

**Mass-Trans** 会区分不同类型的文本并采用不同策略处理：

- 对话框
- 旁白
- 拟声词
- 艺术字
- 注解
- 场景文字
- 线索

### 4. 长篇连续性

对于长篇漫画，**Mass-Trans** 会围绕角色、物品、场景、术语和风格建立项目级记忆，让译文在整个系列中保持一致。

### 5. 低门槛

新手可以按照预设流程快速开始，先得到一个可阅读、可修改的初版结果。通过 Agent 辅助，Mass-Trans 可以让用户用更少步骤完成基础处理。

### 6. 专业调整空间

**Mass-Trans** 不追求完全取代人工。**Mass-Trans** 会输出可继续编辑的结果，包括翻译对照文本和分层图形文件，方便译者、嵌字和修图人员进行后续精修。
