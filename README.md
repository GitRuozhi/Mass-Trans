# Mass-Trans

一个面向 `漫画`和 `复杂静态艺术场景` 翻译和本地化的 AI 工作流。早期版本（MVP）采用 Codex / Claude Code 本地 Skill 的形式实现。

## 愿景

尽可能降低漫画翻译工作的门槛，以及提升专业级工作的上限。

## 特点

- 以整本漫画为基本单位，以分镜作为最小单元，摈弃以 页 为基础的传统工作流。


- 保持角色、物体、场景、术语、画风等漫画专有元素的一致性。


- 针对不同子内容裁取不同的处理策略，页面结构、分镜关系、对话框、拟声词、艺术字、注解、场景文字、线索。


- 针对长篇连续性，围绕角色、信物、场景等专有名词建立项目级Agnet记忆，使得译文在整个系列中保持一致。


- 利用Agnet能力实现自动识别，一个回合完成基本操作，尽可能降低门槛。


- 输出 完整翻译对照文本、分图层图形文件序列，适合专业级精细调整。
